auto-kappa

Auto-kappa is an automation framework for first-principles calculations of anharmonic phonon properties, including thermal conductivity and mode-dependent phonon lifetimes, based on VASP and ALAMODE.

GitHub repository: https://github.com/masato1122/auto-kappa
